# Your work images

Drop your design images into **this folder** (`src/assets/work/`).

- Any `.jpg`, `.jpeg`, `.png`, `.webp` or `.avif` here appears automatically in the
  Studio gallery (`/studio`) — no code changes needed.
- The gallery is **masonry**: it arranges automatically by each image's size
  (tall images stay tall, wide images stay wide).
- They show in **filename order**, so name them `01-...`, `02-...` to control order.
- The `work-01 … work-12` files are placeholders — delete them and add your own.

Then run `npm run build` (or it's picked up automatically in `npm run dev`).
